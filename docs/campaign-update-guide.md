# キャンペーン更新ガイド

キャッシュバックキャンペーンが切り替わったときの更新手順。

---

## 更新が必要なファイル

| ファイル | 変更内容 |
|---|---|
| `public/data/config.json` | チェックボックスのラベル名を変更 |
| `public/data/lenses.json` | 各レンズのキャッシュバック金額を更新 |

---

## 手順

### 1. config.json のラベルを変更

`displayAttributes` 内の `campaignCashbackJpy` の `label` を新しいキャンペーン名に書き換える。

```json
{
  "key": "campaignCashbackJpy",
  "label": "キャンペーン（2026秋）",  ← ここを変更
  "unit": "",
  "format": "campaign"
}
```

### 2. lenses.json の金額を更新

各レンズの `campaignCashbackJpy` フィールドを以下のルールで書き換える。

| 値 | 意味 |
|---|---|
| `数値`（例: `20000`） | キャッシュバック対象。金額を円で記入 |
| `false` | 対象外（ニコン非対象レンズ・他メーカー） |

**例：**
```json
{
  "id": "z-35-1.2-s",
  "priceJpy": 437800,
  "campaignCashbackJpy": 20000,   ← 対象の場合は金額
  ...
},
{
  "id": "z-58-0.95-s-noct",
  "priceJpy": 1138500,
  "campaignCashbackJpy": false,   ← 対象外は false
  ...
}
```

---

## 表示フォーマット

| `campaignCashbackJpy` の値 | 表示 |
|---|---|
| `20000` | `CB: 20,000円` |
| `false` | `×` |
| フィールド未設定 | `×` |
